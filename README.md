# The Room

A lightweight, simple node.js web chatroom server. 

## Installation

You will need [node.js](https://nodejs.org/en/download) installed

### 1. Clone the repo
```sh
git clone git@github.com:NoLilypad/TheRoom.git
cd TheRoom
```

### 2. Install
```sh
npm install
```

### 3. Run the server
```sh
node server.js
```
